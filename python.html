Python

Python je jezik stvoren da bude prikladan za više od jednog stila programiranja. Napravio ga je Guido van Rossum 1991. To je jezik visoke razine, opće namjene i otvorenog koda. Python ima dinamični tip sistema i automatsko upravljanje memorijom.

Programske paradigme koje podržava su: objektno orijentiranje, imperative, funkcionalno i proceduralno programiranje. Python se koristi u: izradi web aplikacija, pri razvijanju softvera i u osiguravanju informacija. (Wikipedia n.d.)
