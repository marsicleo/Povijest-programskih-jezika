Java

Iako mu je prva namjena bila da bude korišten za interaktivni TV projekt, kreator James Gosling je dalje proslijedio da Java postane posve novi programski jezik. Velika značajka Jave je da može jednako funkcionirati na više platformi bez problema.

Java je opće namjene, klasno i objektno orijentiran programski jezik. Koristi se za mrežno programiranje, rađenje web aplikacija, razvoj softvera i grafičkog korisničkog sučelja. Koriste ga: Android OS i njegove aplikacije. (Sebesta 2012)
